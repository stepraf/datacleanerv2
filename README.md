# Data Cleaner v2

A Streamlit application for data cleaning and analysis.

## Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

The app will open in your default web browser.

## Features

- **Import Data**: Upload CSV files and load them into the application
- **Data View**: View and explore your imported data
